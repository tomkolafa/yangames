Rounded pill button for primary actions, used across menus, dialogs and reward screens — soft brand coral with a gentle press-down.

```jsx
<Button variant="primary" size="lg" onClick={play}>Play</Button>
<Button variant="secondary">How to play</Button>
<Button variant="ghost" size="sm">Skip</Button>
```

Variants: `primary` (coral, raised), `secondary` (soft coral tint), `ghost` (text only).
Sizes: `sm` · `md` · `lg`. Props: `fullWidth`, `disabled`, `iconLeft`, `iconRight`.
