Square or circular soft icon button for the game toolbar — menu, help, stats, settings. Wrap a Lucide SVG.

```jsx
<IconButton label="Settings"><Settings size={20}/></IconButton>
<IconButton label="Help" shape="circle" tone="brand"><HelpCircle size={20}/></IconButton>
```

Sizes `sm·md·lg`; shapes `rounded·circle`; tones `soft·plain·brand`.
